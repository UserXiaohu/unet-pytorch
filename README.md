# unet-pytorch
基于pytorch实现的unet图像语义分割算法（数据集天池地表建筑物识别）

* 框架:pytorch
* 网络接口:UNET
* 数据集:天池地表建筑物识别
* 数据集地址以及项目简介:https://tianchi.aliyun.com/competition/entrance/531872/information

详细内容，可以关注微信公众号<是云开源>，回复<unet>进行获取。
